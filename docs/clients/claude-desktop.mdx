# Claude Desktop

<Note>
Claude Desktop supports **local MCP servers only**. It does not support remote MCP connections natively. Use [Claude Code](/clients/claude-code) or [Cursor](/clients/cursor) for remote server access.
</Note>

## Local Server

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "easy-node": {
      "command": "npx",
      "args": ["-y", "@easynodexyz/mcp-x402"]
    }
  }
}
```

## Config File Locations

| OS | Path |
|----|------|
| macOS | `~/Library/Application Support/Claude/claude_desktop_config.json` |
| Windows | `%APPDATA%\Claude\claude_desktop_config.json` |
| Linux | `~/.config/Claude/claude_desktop_config.json` |

## After Configuration

Restart Claude Desktop completely to load the MCP server.
