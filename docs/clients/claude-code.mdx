# Claude Code

## EasyNode X402 Plugin

The EasyNode X402 plugin provides an MCP server with 7 tools for browsing VPS and blockchain node products, purchasing with USDC on Base via x402 payments, managing instances, and decrypting credentials.

### Installing the Plugin

```bash
/plugin marketplace add easynodexyz/mcp-x402
/plugin install easynode-x402-plugin@easynode-marketplace
```

### Setup

The plugin requires `EASYNODE_PRIVATE_KEY` as a **system environment variable**. Claude Code passes it to the remote server via HTTP header ‚Äî it does not read `~/.easy-node/.env`.

Add to your shell profile (`~/.bashrc`, `~/.zshrc`, or `~/.profile`):

```bash
export EASYNODE_PRIVATE_KEY=0x...
```

On Windows, set it via PowerShell or CMD:

```powershell
# PowerShell
[Environment]::SetEnvironmentVariable("EASYNODE_PRIVATE_KEY", "0x...", "User")
```

```cmd
# CMD
setx EASYNODE_PRIVATE_KEY "0x..."
```

Restart Claude Code after setting the variable.

<Note>
The `~/.easy-node/.env` config file is only used by the [local stdio server](#local-server). The plugin uses the remote server.
</Note>

### Configuration

| Variable               | Required | Default | Description                  |
|------------------------|----------|---------|------------------------------|
| `EASYNODE_PRIVATE_KEY` |    Yes   |    ‚Äî    | Wallet private key (`0x...`) |
| `EASYNODE_MAX_PAYMENT` |    No    |  `100`  | Max USDC per transaction     |

### Available Tools

| Tool                  | Description                                       |
|-----------------------|---------------------------------------------------|
| `list_products`       | Browse available VPS and blockchain node products |
| `create_order`        | Purchase a product with USDC via x402 payment     |
| `get_order`           | Check order status and delivery progress          |
| `list_instances`      | List all active instances                         |
| `get_instance`        | View instance details                             |
| `renew_instance`      | Renew an existing instance                        |
| `update_custom_name`  | Rename an instance                                |

## Manual Setup

### Remote Server

Connect to the hosted server ‚Äî no local install needed:

```bash
claude mcp add --transport http \
  easy-node https://mcp-x402.easy-node.xyz/mcp \
  --header "X-Easynode-Private-Key: 0x..."
```

### Local Server

Run the MCP server locally via npx:

```bash
claude mcp add easy-node -- npx -y @easynodexyz/mcp-x402
```

Or add manually to `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "easy-node": {
      "command": "npx",
      "args": ["-y", "@easynodexyz/mcp-x402"]
    }
  }
}
```

### Config File Locations

| OS | Path |
|----|------|
| macOS/Linux | `~/.claude/settings.json` |
| Windows | `%USERPROFILE%\.claude\settings.json` |
