# VS Code + Continue

<Note>
Continue supports **local MCP servers only**. For remote server access, use [VS Code Copilot](/clients/all-clients) or another client with remote support.
</Note>

## Local Server

Add to your Continue configuration (`~/.continue/config.json`):

```json
{
  "experimental": {
    "modelContextProtocolServers": [
      {
        "transport": {
          "type": "stdio",
          "command": "npx",
          "args": ["-y", "@easynodexyz/mcp-x402"]
        }
      }
    ]
  }
}
```

## After Configuration

Restart VS Code to load the MCP server through Continue.
