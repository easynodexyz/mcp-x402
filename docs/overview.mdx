# Easy Node MCP x402

Enable AI assistants to browse, purchase, and manage VPS and blockchain node products using USDC on Base network.

## What is this?

`@easynodexyz/mcp-x402` is a [Model Context Protocol](https://modelcontextprotocol.io/) server that gives AI agents the ability to:

- **Browse** available VPS and blockchain node products
- **Purchase** products using USDC on Base (x402 payment flow)
- **Track** order status and delivery
- **Manage** running instances (list, view details, renew)

The [x402 payment protocol](https://www.x402.org/) is handled automatically — the agent calls tools and payments are processed seamlessly.

## Quick Start

<Steps>
  <Step title="Install & Setup">
    ```bash
    npx @easynodexyz/mcp-x402 setup
    ```
  </Step>
  <Step title="Configure your IDE">
    Add the MCP server to [Claude Desktop](/clients/claude-desktop), [Claude Code](/clients/claude-code), [Cursor](/clients/cursor), or [VS Code](/clients/vscode-continue).
  </Step>
  <Step title="Start using">
    Ask your AI assistant: "List available VPS products" or "Purchase 1 month of VPS-XS"
  </Step>
</Steps>
