EasyNode x402 enable AI assistants to browse, purchase, and manage VPS and blockchain node products using USDC on Base network. Powered by x402 protocol.

## What is this?

`@easynodexyz/mcp-x402` is a [Model Context Protocol](https://modelcontextprotocol.io/) server that gives AI agents the ability to:

- **Browse** available VPS and blockchain node products
- **Purchase** products using USDC on Base (x402 payment flow)
- **Track** order status and delivery
- **Manage** running instances (list, view details, renew)

The [x402 payment protocol](https://www.x402.org/) is handled automatically — the agent calls tools and payments are processed seamlessly.

<Steps>
